 Shopping App UI (Flutter)

A modern and visually rich Shopping App UI built with Flutter, focused on clean layouts, smooth user experience, and responsive design.
This project is UI only and showcases how a real world e-commerce application can look and feel.

Perfect for Flutter beginners to intermediate developers who want to study layouts, widget composition, and UI best practices.
